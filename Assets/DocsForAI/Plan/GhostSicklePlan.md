# GhostSickle 実装プラン

## 目的
GhostSickle を敵中心に反時計回りで回転させ、時間経過で半径が広がる渦巻き挙動を実装する。

## タスク
1. 現状調査: `Assets/Scripts/Projectiles/GhostSickle.cs` と関連コンポーネントの現在の挙動を確認する。
2. 設計: 回転中心を「発射した敵の位置」に保持する方法と、時間依存の半径拡大 + 角速度で軌道を計算するアルゴリズムを決める。
3. 実装: 設計方針に沿って `GhostSickle.cs` を更新し、必要なフィールド・初期化処理・`Update`(または同等)での座標更新を追加する。
4. 調整: パラメータ(初期半径、半径拡大速度、角速度など)をインスペクタ調整可能にし、将来のバランス調整に備える。
5. 動作確認: Unity エディタ上でエラーが出ないこと、位置更新ロジックが意図どおりになることをコード上で確認する。
