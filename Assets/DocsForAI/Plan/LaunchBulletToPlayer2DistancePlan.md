# LaunchBulletToPlayer2 距離判定プラン

## 目的
プレイヤーが遠すぎる場合は弾を発射せず即完了させる距離判定を追加し、インスペクタから閾値を調整できるようにする（0 のときは常時発射）。

## タスク
1. `LaunchBulletToPlayer2` の実装を確認し、発射トリガー前に距離をチェックできる箇所を特定する。
2. 発射抑制用の `SerializeField` 距離パラメータ（デフォルト 8、0 で無効）と Tooltip を追加し、プレイヤー取得失敗時も安全に抜けるようガードを入れる。
3. 距離超過時にアニメーショントリガーを飛ばさず `IsDone` を立てて終了するロジックを実装し、AnimationEvent 側の null アクセスも防ぐ。
4. 想定動作の確認：閾値 0 で常時発射、閾値より遠いときはアニメーション未再生で即完了することをコードレビューで確認する。
